# Nexara Design + Super Skills Brainstorm

## Experience Goal

The site should feel like a premium dual-mode operating system for Tier-2 India:

- Neo: energetic, sharp, builder-facing.
- Trust: professional, structured, decision-maker friendly.
- Same content backbone: Academy, Digital Marketing, Labs.

## Design Stack

- Gateway-first entry with two presentation modes.
- Sticky nav plus route context bar for orientation.
- Large animated hero banners.
- Audience fit, stack details, process, packages, proof, FAQ and intake.
- Super Skills layer to show how the three sections compound.

## Super Skills

These are cross-section capabilities that combine Academy, Digital Marketing and Labs:

- City Launch Stack: skill pipeline + brand system + automation.
- Placement Growth Loop: learner readiness + employer marketing + hiring workflow.
- AI-Ready Business Kit: website clarity + knowledge base + workflow automation.
- Founder Visibility Engine: positioning + content + CRM/AI follow-up.
- Institution Outcome OS: cohort training + reporting + placement dashboards.

## UX Enhancements

- Add a route-aware experience bar below nav.
- Add skip link and focus states.
- Add Super Skills section on Home.
- Add stronger CTA ladder: explore section, view proof, start brief.
- Keep mobile readable with horizontal nav only where needed.

## Architecture Review Additions

- Keep Gateway first and preserve both `#neo/...` and `#trust/...` routes.
- Add Contact as a top-level route instead of hiding enquiry copy inside section CTAs.
- Treat Proof as readiness evidence until public client stories and metrics are approved.
- Use Company for incorporation posture, standards and claim discipline.
- Keep all section expansion inside the shared data model so Neo and Trust stay aligned.

## Next Things To Spin Up On The Website

- Legal-ready footer once entity details are approved: legal name, CIN or LLPIN, registered office, privacy and terms.
- Public case-study layer once client permission exists: challenge, work delivered, outcome, evidence level.
- Downloadable briefs for Academy, Marketing and Labs.
- City launch pages for priority markets such as Visakhapatnam, Vijayawada, Coimbatore and Indore.
- Labs security page expansion with data handling, access control, review and deployment options.
